# CLAUDE.md

A curated configuration file with opinionated guidelines for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) development workflows.

## What is CLAUDE.md?

CLAUDE.md is a configuration file that Claude Code reads to understand project-specific guidelines, conventions, and rules. It lives in your project root or home directory for global settings.

## Usage

**Global installation (applies to all projects):**
```bash
cp CLAUDE.md ~/.claude/CLAUDE.md
```

**Per-project installation:**
```bash
cp CLAUDE.md /path/to/your/project/CLAUDE.md
```

## Customization

Fork this repo and modify the rules to match your workflow.
